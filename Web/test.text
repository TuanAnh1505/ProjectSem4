1. Users
GET /users: Lấy tất cả người dùng.
GET /users/1: Lấy thông tin người dùng với UserID = 1.
POST /users: Tạo người dùng mới.
{
    "FullName": "Pham Thi D",
    "Email": "phamthid@example.com",
    "Phone": "0971234567",
    "Address": "101 Duong GHI, Hue",
    "IsActive": 1
}
PUT /users/1: Cập nhật thông tin người dùng với UserID = 1.
 json
{
    "FullName": "Nguyen Van A Updated",
    "Phone": "0909999999"
}
DELETE /users/1: Xóa người dùng với UserID = 1.
2. Tours

GET /tours/1: Lấy thông tin tour với TourID = 1.
POST /tours: Tạo tour mới.
